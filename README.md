# Guld & Silver Analysverktyg

Ett Python-program som analyserar relationen mellan guld- och silverpriser och hjälper användaren att beräkna köpkraft baserat på en budget.

## Funktioner
- Hämtar aktuella priser (simulerad data).
- Beräknar "Gold-to-Silver Ratio".
- Räknar ut hur mycket metall du kan köpa för dina pengar.
- Loggar händelser till fil och konsol.
